- Responsive design

## Running Locally

Clone the project

```bash
  git clone https://link-to-the-project
```

Navigate to the project directory

```bash
  cd my-project
```

Install the dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Repeat these steps for both frontend and backend files.
